Калькулятор затмений — это удобный веб-инструмент, который
позволяет быстро узнать астрономическую информацию о любой
выбранной дате: солнечные и лунные затмения, лунные фазы
(новолуния), лунный день, лунации по Брауну и Миусу, а также аркан дня
по системе Таро. Кроме того, он показывает ближайшие даты важных
астрономических циклов (Метон, Сарос, Экселигмос и др.).
Для чего создан калькулятор?
Он помогает:
● быстро проверить, было ли затмение в определенный день (или
ближайшие к нему),
● понять, в каком лунном дне и лунации находится выбранная дата,
● увидеть, через сколько дней наступит следующее новолуние,
● сориентироваться в долгосрочных астрономических циклах,
● получить эзотерическую интерпретацию дня через аркан Таро.
